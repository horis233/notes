### font-family
font-family可以设置多个值，这样能保证前面的字体如果没有被找到，可以直接使用后面的字体
并且还可以在最后一项指定一个通用字体，比如`sanrif, sans-serif`

### font-size
两种单位：一种是像素或者点，另外一种是相对单位em或者rem

### 文本属性
- text-indent 文本缩进
- letter-spacing 字符间距
- word-spacing 单词间距
- text-decoration 文本装饰
- text-align 文本对其
- line-height 行高
- text-transform 文本转换
	+ none
	+ upppercase 全部大写
	+ lowercase  全部小写
	+ capitalize 首字母大写
- vertical-align 垂直对齐
