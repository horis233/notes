### 5.1 表达式语句

```js
delete obj.property;
```

### 5.2 复合语句和空语句
多条语句联合在一个形成复合语句

```js
{
    x = Math.PI;
    cx = Math.cos(x);
}
```

### 5.3 声明语句
`var`和`function`
全局变量是全局对象的属性。var声明的变量是无法通过delete删除的
函数的声明只能出现在所嵌套的函数的顶部。也就是说函数的声明不能出现在if语句,while语句或其他任何语句中

### 5.4 条件语句
`switch`是按照`===`比较的
`return`和`break`都可以用于终止`switch`循环

### 5.5 循环
`for...in`用于遍历对象，但是其只遍历可枚举的属性
主流的浏览器厂商的js实现是按照属性定义的先后顺序来枚举简单对象的。如果使用对象直接量，则按照直接量中属性出现的顺序枚举。

### 5.6 跳转
从一个位置跳到另一个位置
- continue 结束本次循环，开始下一次循环
- break 跳出循环
- return
- throw

当`break`和标签一起使用时，程序将跳转到这个标签所标志的语句块的结束。
需要注意的是，不管break语句带不带标签，它的控制权都无法越过函数。也就是不能跳到函数外部。

```js
// 结束loopname这个循环
break loopname;
```

`continue`语句同样也可以带有标签

```js
// 结束本次循环
continue loopname;
```

### 5.7 其他语句类型
严格模式：
- 严格模式禁止使用`with`语句
- 严格模式中，所有的变量都要事先声明
- 调用函数中(非对象方法)中的`this`时，会抛出一个错误
- 严格模式中，给只读属性赋值和给不可扩展属性创建新成员都将抛出一个错误。
- 禁止使用`arguments.caller`和`arguments.callee`
